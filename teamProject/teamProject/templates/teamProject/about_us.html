Aim of the game

Have you been stuck inside for almost a year now beacause of lockdown? Well your answer better be yes becasye we sure have! This game is designed to make you feel like you get can outside and virtually break the 5km restrictions! You are thrown into a random place in Cork and you must guess where that is. We have 3 gamemodes: Easy ( for kids ), Retro ( for the rents or if you're feeling a bit ambitious ) and finally Hard mode if you're a pro. The Easy and Retro mode are simple MCQ games, whereas the Hard mode provides you with a map where you must pinpoint where you are on it.

Scoring

Each gamemode consists of 5 rounds. The maximum amount points you can obtain each round is 150. Therefore the mamimum amount of points you are able to achieve is 750!!
